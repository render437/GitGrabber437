# GitGrabber437

This repository is designed to be a tool for fetching and displaying the code of specific files from GitHub repositories without cloning the entire repo.

## Setup

1. Clone this repository to your local machine.
2. Make the `gitgrabber437.sh` script executable: `chmod +x gitgrabber437.sh`

## Usage

Run the script, and it will prompt you for the GitHub repository URL and the path to the specific file you want to view.

```bash
./gitgrabber437.sh
